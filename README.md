# instructions

1.  clone repository

    ```bash
    git clone https://github.com/ingriddorioschulze/cinnamoon
    cd cinnamoon
    ```

2.  install dependencies

    ```bash
    npm install
    ```

3.  start the application on your local machine

    ```bash
    npm start
    ```
